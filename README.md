# Playwright QA Automation Portfolio

Automated test suite built with Playwright and TypeScript.

## Project 1 — Login Test Automation

Automated login tests for [Practice Test Automation](https://practicetestautomation.com/practice-test-login/).

### Test Scenarios
- ✅ Successful login with valid credentials
- ❌ Failed login with incorrect password
- ❌ Failed login with incorrect username

### Tech Stack
- [Playwright](https://playwright.dev/)
- TypeScript
- Page Object Model (POM)
